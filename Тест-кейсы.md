# Test Cases

## 1) Создание задачи
Steps:
1. Нажать "Создать задачу"
2. Ввести название
3. Нажать "Сохранить"
   
Expected Result: Задача отображается в списке

## 2) Создание задачи без названия
Steps:
1. Нажать "Создать задачу"
2. Оставить поле пустым
3. Нажать "Сохранить"
   
Expected Result: Появляется сообщение об ошибке

## 3) Изменение названия задачи
Steps:
1. Выбрать задачу
2. Изменить название
3. Сохранить
   
Expected Result: Новое название отображается

## 4) Изменение статуса задачи
Steps:
1. Выбрать задачу
2. Изменить статус на Done
   
Expected Result: Статус сохранён

## 5) Удаление задачи
Steps:
1. Выбрать задачу
2. Нажать "Удалить"
   
Expected Result: Задача исчезает из списка
