# Test Cases

## 1) Создание задачи
Steps:
1. Нажать "Создать задачу"
2. Ввести название
3. Нажать "Сохранить"
   
Expected: Задача отображается в списке

## 2) Создание задачи без названия
Steps:
1. Нажать "Создать задачу"
2. Оставить поле пустым
3. Нажать "Сохранить"
   
Expected: Появляется сообщение об ошибке

## 3) Изменение задачи
Steps:
1. Выбрать задачу
2. Изменить название
3. Сохранить
   
Expected: Новое название отображается

## 4) Удаление задачи
Steps:
1. Выбрать задачу
2. Нажать "Удалить"
   
Expected: Задача исчезает из списка
