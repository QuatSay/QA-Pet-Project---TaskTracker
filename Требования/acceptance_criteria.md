# Acceptance criteria

## 1) Создание задачи
- Название задачи обязательно
- Максимальная длина названия — 255 символов
- Статус задачи по умолчанию — New
- Задача отображается в списке после сохранения

## 2) Редактирование задачи
- Пользователь может изменить название задачи
- Изменения сохраняются после обновления страницы

## 3) Изменение статуса задачи
- Доступные статусы: New, In Progress, Done
- Статус сохраняется корректно

## 4) Удаление задачи
- Пользователь может удалить задачу
- Удалённая задача не отображается в списке
