
## 1) Регистрация пользователя
- Email обязателен
- Email должен быть уникальным
- Пароль обязателен
- Минимальная длина пароля — 8 символов
- После успешной регистрации пользователь может авторизоваться


## 2) Авторизация пользователя
- Пользователь может авторизоваться с валидными данными
- При невалидных данных отображается ошибка
- После авторизации пользователь перенаправляется на страницу задач


## 3) Создание задачи
- Название задачи обязательно
- Максимальная длина названия — 255 символов
- Статус задачи по умолчанию — New
- Задача отображается в списке после сохранения


## 4) Редактирование задачи
- Пользователь может изменить название задачи
- Изменения сохраняются после обновления страницы


## 5) Изменение статуса задачи
- Доступные статусы: New, In Progress, Done
- Статус сохраняется корректно


## 6) Удаление задачи
- Пользователь может удалить задачу
- Удалённая задача не отображается в списке
