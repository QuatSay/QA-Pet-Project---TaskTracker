# User Stories

## 1) Создание задачи
Как пользователь,
я хочу создавать задачи,
чтобы управлять своим списком дел.


## 2) Редактирование задачи
Как пользователь,
я хочу редактировать существующую задачу,
чтобы актуализировать информацию.


## 3) Изменение статуса задачи
Как пользователь,
я хочу изменять статус задачи,
чтобы отслеживать прогресс выполнения.


## 4) Удаление задачи
Как пользователь,
я хочу удалять задачу,
чтобы очищать список от неактуальных задач.
