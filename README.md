# QA Pet Project - TaskTracker

Учебный QA-проект для отработки базовых навыков функционального и API-тестирования веб-приложения.

Цель: практиковаться в написании тест-кейсов, чек-листов, баг-репортов, работе с API, SQL и Chrome DevTools.

Инструменты:
- Postman
- Chrome DevTools
- SQL 
- Markdown / GitHub

Структура репозитория:
- Требования
- Тест-дизайн
- Чек-лист
- Тест-кейсы
- Баг-репорты
- API тестирование 
- SQL
- DevTools
