# Checklist

- Можно создать задачу с названием
- Нельзя создать задачу без названия
- Название >255 символов не принимается
- Задача отображается в списке
- Статус задачи по умолчанию — New
- Можно изменить название задачи
- Можно изменить статус задачи
- Можно удалить задачу
- Данные сохраняются после обновления страницы
