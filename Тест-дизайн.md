# Test Strategy 


## Цель тестирования
Обеспечить базовое качество веб-приложения TaskTracker путём функционального и API-тестирования ключевых пользовательских сценариев.


## Объект тестирования
- Веб-приложение TaskTracker
- REST API
- Работа с базой данных (на уровне проверки данных)


## Уровни тестирования
- System Testing
- API Testing


## Виды тестирования
- Functional testing
- Smoke testing
- Regression testing
- API testing


## Инструменты
- Браузер: Google Chrome
- Chrome DevTools
- Postman
- SQL client (SQLite / PostgreSQL)
- Google Docs / Markdown


## Завершение тестирования
- Все smoke-тесты пройдены
- Критические дефекты исправлены
